    # 使用

    由于组件库使用的是 vue2.7 ，使用时请确保项目已经升级至最新的 vue2.7 版本。

    为减少打包资源，组件库并未提供第三方库（element-ui 或 vxe-table）的组件引入，使用时请自行引入组件内对应的第三方组件。

    # 全局引入

    import VinUi from 'vin-ui';

Vue.use(VinUi);

    import Vue from 'vue';
