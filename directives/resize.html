# 元素尺寸监听指令

# 基础用法

使用 v-resize 指令可以监听元素的尺寸变化。

宽度: 0px
高度: 0px
拖动右下角调整大小
<template>
  <div class="example-resize">
    <div 
      class="resize-box" 
      v-resize="handleResize"
    >
      <div class="size-info">
        宽度: {{ width }}px<br>
        高度: {{ height }}px
      </div>
    </div>
    <div class="tip">拖动右下角调整大小</div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      width: 0,
      height: 0
    }
  },
  methods: {
    handleResize({ width, height }) {
      this.width = Math.round(width)
      this.height = Math.round(height)
    }
  }
}
</script>

<style scoped>
.example-resize {
  padding: 20px;
}
.resize-box {
  width: 300px;
  height: 200px;
  border: 1px solid #dcdfe6;
  resize: both;
  overflow: auto;
  padding: 20px;
  position: relative;
}
.size-info {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  color: #666;
  line-height: 1.8;
}
.tip {
  margin-top: 15px;
  color: #909399;
  font-size: 14px;
}
</style>

# 立即触发

使用 immediate 修饰符可以在初始化时立即触发回调。

宽度: 0px
高度: 0px
初始化时会立即触发回调
<template>
  <div class="example-resize">
    <div 
      class="resize-box" 
      v-resize.immediate="handleResize"
    >
      <div class="size-info">
        宽度: {{ width }}px<br>
        高度: {{ height }}px
      </div>
    </div>
    <div class="tip">初始化时会立即触发回调</div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      width: 0,
      height: 0
    }
  },
  methods: {
    handleResize({ width, height }) {
      this.width = Math.round(width)
      this.height = Math.round(height)
    }
  }
}
</script>
