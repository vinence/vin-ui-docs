# 数据脱敏指令

# 基础用法

使用 v-desensitize 指令可以快速对敏感数据进行脱敏处理。

<template>
  <el-form class="example-desensitize-form">
    <el-form-item label="电话号码:">
      <span v-desensitize.tooltip="{ value: phone, type: 'phone' }" />
    </el-form-item>
    <el-form-item label="电子邮件:">
      <span v-desensitize:email="'example@domain.com'"></span>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      phone: '13800138000'
    };
  }
};
</script>
<style scoped>
.example-desensitize-form {
  padding: 20px;
}

.el-form-item {
  margin-bottom: 15px;
}
</style>

# 自定义规则

可以通过传入对象来自定义脱敏规则。

<template>
  <div class="example-desensitize">
    <div class="item">
      <span v-desensitize="{
        value: '自定义脱敏规则',
        rule: { start: 2, end: 2, mask: '#' }
      }"></span>
    </div>
    <div class="item">
      <span v-desensitize="{
        value: '12345678901',
        rule: {
          pattern: /(\d{3})(\d+)(\d{4})/,
          formatter: (m) => `${m[1]}-***-${m[3]}`
        }
      }"></span>
    </div>
  </div>
</template>

<style scoped>
.example-desensitize {
  padding: 20px;
}
.item {
  margin-bottom: 15px;
}
</style>

# 指令参数

参数 说明 类型 可选值 默认值
value 脱敏类型或配置对象 string/object phone/email/idCard/bankCard/name/address phone
type 脱敏类型（作为参数传入） string phone/email/idCard/bankCard/name/address -
rule 自定义脱敏规则 object - -

# 自定义规则配置

参数 说明 类型 默认值
start 保留开始位置的字符数 number 0
end 保留结束位置的字符数 number 0
mask 用于替换的掩码字符 string *
pattern 自定义匹配模式 RegExp -
formatter 自定义格式化函数 Function -
