# Tooltip 文字提示

基于 Element UI 的 Tooltip 组件进行封装,可以根据内容是否溢出自动显示文字提示。主要用于解决文本溢出显示问题。

# 功能特性

  • 自动判断 - 根据内容是否溢出自动显示/隐藏 tooltip
  • 响应式更新 - 支持窗口大小变化时重新判断
  • 主题定制 - 支持 light/dark 两种主题
  • 位置调整 - 支持 12 个方向的展示位置
  • 宽度控制 - 支持自定义 tooltip 宽度
  • 样式定制 - 支持自定义 popper 类名

# 基础用法

当文本内容超出显示范围时,自动显示 tooltip。

文本未超出时不显示:
文本超出时显示:

通过 text 属性设置显示的文本内容。只有当文本溢出时才会显示 tooltip。

<template>
  <div>
    <div class="demo-row">
      <span>文本未超出时不显示:</span>
      <div style="width: 200px">
        <vin-tooltip text="这是一段较短的文本"></vin-tooltip>
      </div>
    </div>

    <div class="demo-row">
      <span>文本超出时显示:</span>
      <div style="width: 200px">
        <vin-tooltip 
          text="这是一段非常长的文本,超出显示范围时会自动显示tooltip来展示完整内容">
        </vin-tooltip>
      </div>
    </div>
  </div>
</template>

# 展示位置

Tooltip 支持 12 个不同方向的展示位置。

左上提示
左中提示
左下提示
右上提示
右中提示
右下提示

使用 placement 属性设置展示位置:

  • top/top-start/top-end: 顶部显示
  • bottom/bottom-start/bottom-end: 底部显示
  • left/left-start/left-end: 左侧显示
  • right/right-start/right-end: 右侧显示

# 自定义样式

支持自定义 tooltip 的样式。

自定义样式:
自定义主题:

通过 popper-class 属性添加自定义类名:

  • 可以覆盖默认样式
  • 支持主题定制
  • 注意样式优先级
<template>
  <div>
    <div class="demo-row">
      <span>自定义样式:</span>
      <div style="width: 150px">
        <vin-tooltip 
          text="这是一段自定义样式的提示文本"
          popper-class="custom-tooltip"
          effect="dark">
        </vin-tooltip>
      </div>
    </div>

    <div class="demo-row">
      <span>自定义主题:</span>
      <div style="width: 150px">
        <vin-tooltip 
          text="这是一段使用自定义主题的提示文本"
          popper-class="custom-theme-tooltip">
        </vin-tooltip>
      </div>
    </div>
  </div>
</template>

# API

# Props

参数 说明 类型 可选值 默认值
text 显示的文本内容 string/number - ''
effect tooltip 主题 string dark/light light
placement tooltip 出现位置 string top/top-start/top-end/bottom/bottom-start/bottom-end/left/left-start/left-end/right/right-start/right-end bottom-start
popover-width tooltip 宽度 string - 200px
popper-class 自定义类名 string - ''
disabled 是否禁用 tooltip boolean - false
offset 出现位置的偏移量 number - 0
transition 定义渐变动画 string - el-fade-in-linear
visible-arrow 是否显示箭头 boolean - true
popper-options popper.js 的参数 object - { boundariesElement: 'body', gpuAcceleration: false }
enterable 鼠标是否可进入 tooltip boolean - true
hide-after 延迟关闭(单位:毫秒) number - 0
tabindex Tooltip 组件的 tabindex number - 0

# Events

事件名 说明 参数
show 显示时触发 -
hide 隐藏时触发 -
after-enter 显示动画播放完毕后触发 -
after-leave 隐藏动画播放完毕后触发 -

# Slots

插槽名 说明
default 触发 tooltip 显示的 HTML 元素

# 指令

指令名 说明 参数
v-auto-show-text 自动判断文本是否溢出并显示 tooltip -
v-resize 监听元素大小变化 回调函数

# 使用建议

  1. 文本溢出判断

    • 组件会自动判断文本是否溢出
    • 只有在文本溢出时才会显示 tooltip
    • 支持窗口大小变化时重新判断
    • 建议设置合适的容器宽度

  2. 样式定制

    • 使用 popper-class 添加自定义样式
    • 可以通过 CSS 变量调整主题样式
    • 注意样式优先级和层级管理
    • 避免与其他弹出层组件冲突

  3. 性能优化

    • 避免频繁更新文本内容
    • 合理设置 tooltip 宽度
    • 使用 hide-after 控制自动关闭
    • 注意内存泄漏问题

  4. 无障碍访问

    • 设置合适的 tabindex
    • 提供键盘操作支持
    • 考虑屏幕阅读器用户

# 常见问题

  1. tooltip 不显示

    • 检查文本是否真的溢出
    • 确认容器宽度设置正确
    • 查看是否被其他元素遮挡

  2. 样式异常

    • 检查 z-index 设置
    • 确认自定义样式优先级
    • 查看是否与其他组件样式冲突

  3. 性能问题

    • 避免在循环中使用
    • 控制更新频率
    • 及时销毁不需要的实例
