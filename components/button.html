# Button 按钮

VinButton 是基于 Element UI Button (opens new window) 组件的二次封装，完全继承了 Element UI Button 的所有功能和 API。您可以像使用 Element UI Button 一样使用 VinButton，同时获得了额外的安全性增强。

TIP

关于按钮的基础用法，请参考 Element UI Button 文档 (opens new window)。本文档主要介绍 VinButton 的增强特性。

# 基础用法

最基础的按钮用法。

安全提醒

尝试以下步骤来体验按钮的安全性：

  1. 按 F12 打开浏览器开发者工具
  2. 在元素面板中找到禁用按钮
  3. 尝试删除 disabled 属性
  4. 观察按钮的行为变化

即使移除了 disabled 属性，VinButton 依然能保持禁用状态，这体现了组件的安全性设计。

# 防抖与节流

通过设置 mode 和 time 属性来控制按钮的点击行为，有效防止重复提交或限制提交频率。

# 基础用法

点击次数: 0
防抖按钮 节流按钮

提示: 快速点击按钮查看效果

  • 防抖按钮: 在最后一次点击1秒后才会触发
  • 节流按钮: 每1秒最多只能触发一次

通过 mode 属性设置按钮的特殊点击模式，time 属性设置时间间隔。

<template>
  <div class="button-demo">
    <div class="demo-row">
      <span class="demo-label">点击次数: {{ count }}</span>
      <div class="demo-buttons">
        <!-- 防抖模式 -->
        <vin-button 
          type="primary" 
          mode="debounce" 
          time="1s"
          @click="handleClick"
        >
          防抖按钮
        </vin-button>
        <!-- 节流模式 -->
        <vin-button 
          type="success" 
          mode="throttle" 
          time="1s"
          @click="handleClick"
        >
          节流按钮
        </vin-button>
      </div>
    </div>
    <div class="demo-tips">
      <p>提示: 快速点击按钮查看效果</p>
      <ul>
        <li>防抖按钮: 在最后一次点击1秒后才会触发</li>
        <li>节流按钮: 每1秒最多只能触发一次</li>
      </ul>
    </div>
  </div>
</template>

<script>
export default {
  name: 'ButtonThrottle',
  data() {
    return {
      count: 0
    }
  },
  methods: {
    handleClick() {
      this.count++
    }
  }
}
</script>

# 高级用法

点击次数: 0
延迟防抖 自定义节流

提示: 快速点击按钮查看效果

  • 延迟防抖: 停止点击1秒后才会触发
  • 自定义节流: 首次点击延迟1秒后执行，之后每1秒最多执行一次

可以通过 immediate 和 trailing 属性进一步控制按钮的行为。

# 属性说明

参数 说明 类型 可选值 默认值
mode 特殊点击模式 string debounce / throttle -
time 防抖或节流的时间间隔 string - 1s
immediate 是否立即执行 boolean - true
trailing 是否在结束时执行（仅节流模式有效） boolean - true

# 使用场景

  1. 防抖模式（debounce）
  • 适用于搜索输入、表单提交等场景
  • 连续触发时，只在最后一次触发后的指定时间执行
  • immediate=true: 首次点击立即执行，之后的点击防抖
  • immediate=false: 等待停止点击指定时间后才执行
  1. 节流模式（throttle）
  • 适用于频繁点击、快速提交等场景
  • 在指定时间内只执行一次
  • immediate=true: 首次点击立即执行
  • immediate=false: 等待指定时间后执行
  • trailing=true: 最后一次点击会被执行
  • trailing=false: 最后一次点击可能被忽略

# 注意事项

  1. 时间格式支持：
  • 秒：1s、0.5s
  • 毫秒：1000ms、500ms
  1. 防抖模式特点：
  • 连续点击时会重置计时器
  • 适合需要等待用户操作完成的场景
  • immediate 属性影响首次点击的行为
  1. 节流模式特点：
  • 固定时间内只执行一次
  • 适合限制操作频率的场景
  • immediate 和 trailing 属性可以精确控制执行时机

# API

# 属性

参数 说明 类型 可选值 默认值
mode 特殊点击模式，debounce：防抖，throttle：节流 string throttle/debounce —
time 防抖或节流的时间间隔 string — 1s
size 尺寸 string medium / small / mini —
type 类型 string primary / success / warning / danger / info / text —
plain 是否朴素按钮 boolean — false
round 是否圆角按钮 boolean — false
circle 是否圆形按钮 boolean — false
loading 是否加载中状态 boolean — false
disabled 是否禁用状态 boolean — false
icon 图标类名 string — —
autofocus 是否默认聚焦 boolean — false
native-type 原生 type 属性 string button / submit / reset button

# 事件

事件名称 说明 回调参数
click 点击触发 (event: Event)

# 插槽

插槽名 说明
default 按钮的内容
icon 自定义图标内容
