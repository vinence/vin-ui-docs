# ResizeObserver 元素尺寸监听

VinResizeObserver 是一个用于监听元素尺寸变化的组件，支持所有现代浏览器和 IE11。

TIP

组件使用 object 标签实现尺寸监听，相比 ResizeObserver API 具有更好的浏览器兼容性。

# 基础用法

基础的元素尺寸监听用法。

使用说明

拖动右下角调整元素大小，观察尺寸变化通知：

尺寸变化次数: 0

上次变化时间: -

通过 notify 事件获取元素尺寸变化通知。

<template>
  <div class="resize-demo">
    <div class="tip-section">
      <div class="tip-title">
        <i class="el-icon-info"></i>
        <span>使用说明</span>
      </div>
      <div class="tip-content">
        <p>拖动右下角调整元素大小，观察尺寸变化通知：</p>
      </div>
    </div>

    <div class="resize-container" ref="container">
      <resize-observer @notify="handleResize" />
      <div class="resize-content">
        <p>尺寸变化次数: {{ resizeCount }}</p>
        <p>上次变化时间: {{ lastResizeTime }}</p>
      </div>
    </div>
  </div>
</template>

<script>
export default {
  name: 'BasicResizeObserver',
  data() {
    return {
      resizeCount: 0,
      lastResizeTime: '-'
    };
  },
  methods: {
    handleResize() {
      this.resizeCount++;
      this.lastResizeTime = new Date().toLocaleTimeString();
    }
  }
};
</script>

# 实时尺寸展示

实时展示元素的尺寸变化。

实时尺寸

拖动右下角调整元素大小，实时查看尺寸变化：

宽度: 0px
高度: 0px
0 x 0

结合 notify 事件实时获取并展示元素尺寸。

<template>
  <div class="resize-realtime">
    <div class="tip-section">
      <div class="tip-title">
        <i class="el-icon-info"></i>
        <span>实时尺寸</span>
      </div>
      <div class="tip-content">
        <p>拖动右下角调整元素大小，实时查看尺寸变化：</p>
      </div>
    </div>

    <div class="resize-container" ref="container">
      <resize-observer @notify="handleResize" />
      <div class="resize-content">
        <div class="size-info">
          <div class="size-item">
            <span class="label">宽度:</span>
            <span class="value">{{ width }}px</span>
          </div>
          <div class="size-item">
            <span class="label">高度:</span>
            <span class="value">{{ height }}px</span>
          </div>
        </div>
        <div class="size-visual" :style="{ width: width + 'px', height: height + 'px' }">
          {{ width }} x {{ height }}
        </div>
      </div>
    </div>
  </div>
</template>

<script>
export default {
  name: 'RealtimeResizeObserver',
  data() {
    return {
      width: 0,
      height: 0,
      resizeTimer: null
    };
  },
  mounted() {
    this.$nextTick(this.updateSize);
  },
  beforeDestroy() {
    if (this.resizeTimer) {
      clearTimeout(this.resizeTimer);
    }
  },
  methods: {
    handleResize() {
      if (this.resizeTimer) {
        clearTimeout(this.resizeTimer);
      }
      this.resizeTimer = setTimeout(this.updateSize, 16);
    },
    updateSize() {
      const container = this.$refs.container;
      if (container) {
        const newWidth = container.offsetWidth;
        const newHeight = container.offsetHeight;
        if (this.width !== newWidth || this.height !== newHeight) {
          this.width = newWidth;
          this.height = newHeight;
        }
      }
    }
  }
};
</script>

# Events

事件名称 说明 回调参数
notify 元素尺寸发生变化时触发 -

# 注意事项

  1. 组件会在父元素中创建一个不可见的监听层
  2. 监听层不会影响父元素的布局和交互
  3. 组件会自动处理 IE 浏览器的兼容性问题
