# TreeSelect 树形选择器

VinTreeSelect 是基于 Element UI Select (opens new window) 和 Element UI Tree (opens new window) 组件封装的树形选择器组件，支持单选和多选模式。

TIP

组件继承了 Element UI Select 和 Tree 的基础功能，并增加了全选、反选等便捷操作。

# 基础用法

基础的树形选择器用法。

功能说明

树形选择器支持以下特性：

  1. 单选/多选模式
  2. 支持搜索过滤
  3. 支持全选/反选操作
单选模式：
多选模式：

通过 options 属性传入树形数据，multiple 属性控制单选/多选模式。

<template>
  <div class="tree-select-demo">
    <div class="tip-section">
      <div class="tip-title">
        <i class="el-icon-info"></i>
        <span>功能说明</span>
      </div>
      <div class="tip-content">
        <p>树形选择器支持以下特性：</p>
        <ol>
          <li>单选/多选模式</li>
          <li>支持搜索过滤</li>
          <li>支持全选/反选操作</li>
        </ol>
      </div>
    </div>

    <div class="select-group">
      <div class="select-item">
        <div class="item-title">单选模式：</div>
        <vin-tree-select
          v-model="singleValue"
          :options="options"
          placeholder="请选择"
        />
      </div>
      <div class="select-item">
        <div class="item-title">多选模式：</div>
        <vin-tree-select
          v-model="multipleValue"
          :options="options"
          multiple
          placeholder="请选择"
        />
      </div>
    </div>
  </div>
</template>

            {

                { value: '2-1-1', label: '三级 2-1-1' }
              ]

            {
              value: '2-2',

              children: [
                { value: '2-2-1', label: '三级 2-2-1' }
              ]
            }
          ]
        }
      ]
    };
  }
};
</script>

# 自定义配置

可以自定义字段名称和按钮文字。

自定义配置

支持以下自定义配置：

  1. 自定义数据字段映射
  2. 自定义按钮文字
  3. 控制操作按钮显示

通过 treeProps 配置数据字段映射，通过 checkAllText 等属性自定义按钮文字。

<template>
  <div class="tree-select-custom">
    <div class="tip-section">
      <div class="tip-title">
        <i class="el-icon-info"></i>
        <span>自定义配置</span>
      </div>
      <div class="tip-content">
        <p>支持以下自定义配置：</p>
        <ol>
          <li>自定义数据字段映射</li>
          <li>自定义按钮文字</li>
          <li>控制操作按钮显示</li>
        </ol>
      </div>
    </div>

    <div class="select-group">
      <vin-tree-select
        v-model="value"
        :options="options"
        :tree-props="customProps"
        :check-all-text="'全部'"
        :reset-text="'重置'"
        :reverse-check-text="'反向选择'"
        multiple
        placeholder="请选择"
      />
    </div>
  </div>
</template>

<script>
export default {
  name: 'CustomTreeSelect',
  data() {
    return {
      value: [],
      customProps: {
        value: 'id',
        label: 'name',
        children: 'subList'
      },
      options: [
        {
          id: '1',
          name: '自定义 1',
          subList: [
            {
              id: '1-1',
              name: '自定义 1-1',
              subList: [
                { id: '1-1-1', name: '自定义 1-1-1' }
              ]
            }
          ]
        },
        {
          id: '2',
          name: '自定义 2',
          subList: [
            {
              id: '2-1',
              name: '自定义 2-1',
              subList: [
                { id: '2-1-1', name: '自定义 2-1-1' }
              ]
            }
          ]
        }
      ]
    };
  }
};
</script>

# API

# Props

参数 说明 类型 可选值 默认值
defaultValue 多选默认值数组 array - []
defaultData 单选默认值对象 object - {}
checkAllText 全选按钮文字 string - '全选'
resetText 清空按钮文字 string - '清空'
reverseCheckText 反选按钮文字 string - '反选'
options 可选项数据数组 array - []
treeProps 配置选项 object - { value: 'value', label: 'label', children: 'children' }
checkBoxBtn 是否显示操作按钮 boolean - true
multiple 是否多选 boolean - false
width 选择框宽度 string - '100%'

# Events

事件名称 说明 回调参数
handleNodeClick 节点被点击时触发 单选模式: {id, label}, 多选模式: 选中节点数组

# Methods

方法名 说明 参数
setDefaultValue 设置默认值 defaultData: {id, label}
clearAll 清空选择 -
