# Desensitize 脱敏组件

基于 Element UI 的 Input 组件进行封装，提供多种数据脱敏展示方案，支持手机号、银行卡号、身份证号、邮箱等类型的数据脱敏处理。

# 基础用法

通过 s-type 属性指定脱敏类型。

当前值: 13812345678
当前值: 6222021234567890123
当前值: 310101199001011234
当前值: example@domain.com
当前值: 这是一段测试文本

使用 s-type 属性来选择不同的脱敏类型。支持的类型包括：

  • phone: 手机号脱敏
  • bank: 银行卡号脱敏
  • idCard: 身份证号脱敏
  • email: 邮箱地址脱敏
  • all: 全部字符脱敏
<template>
  <div class="desensitize-demo">
    <el-form label-width="120px">
      <!-- 手机号脱敏 -->
      <el-form-item label="手机号">
        <vin-desensitize 
          v-model="form.phone" 
          s-type="phone" 
          placeholder="请输入手机号"
        />
        <span class="demo-value">当前值: {{ form.phone }}</span>
      </el-form-item>

      <!-- 银行卡号脱敏 -->
      <el-form-item label="银行卡号">
        <vin-desensitize 
          v-model="form.bank" 
          s-type="bank" 
          placeholder="请输入银行卡号"
        />
        <span class="demo-value">当前值: {{ form.bank }}</span>
      </el-form-item>

      <!-- 身份证号脱敏 -->
      <el-form-item label="身份证号">
        <vin-desensitize 
          v-model="form.idCard" 
          s-type="idCard" 
          placeholder="请输入身份证号"
        />
        <span class="demo-value">当前值: {{ form.idCard }}</span>
      </el-form-item>

      <!-- 邮箱脱敏 -->
      <el-form-item label="邮箱地址">
        <vin-desensitize 
          v-model="form.email" 
          s-type="email" 
          placeholder="请输入邮箱地址"
        />
        <span class="demo-value">当前值: {{ form.email }}</span>
      </el-form-item>

      <!-- 全部脱敏 -->
      <el-form-item label="全部脱敏">
        <vin-desensitize 
          v-model="form.all" 
          s-type="all" 
          placeholder="请输入任意内容"
        />
        <span class="demo-value">当前值: {{ form.all }}</span>
      </el-form-item>
    </el-form>
  </div>
</template>

<script>
export default {
  name: 'DesensitizeBasic',
  data() {
    return {
      form: {
        phone: '13812345678',
        bank: '6222021234567890123',
        idCard: '310101199001011234',
        email: 'example@domain.com',
        all: '这是一段测试文本'
      }
    }
  }
}
</script>

# 禁用状态

禁用状态下可以控制是否允许查看完整信息。

点击右侧图标可切换显示完整信息
禁止查看完整信息，不显示切换按钮

使用 disabled 属性设置禁用状态，disabled-show 属性控制禁用状态下是否可以查看完整信息。

<template>
  <div class="desensitize-demo">
    <el-form label-width="200px">
      <!-- 允许查看完整信息 -->
      <el-form-item label="禁用状态-允许查看">
        <vin-desensitize
          v-model="form.value1"
          s-type="phone"
          disabled
          :disabled-show="true"
        />
        <div class="demo-tip">
          点击右侧图标可切换显示完整信息
        </div>
      </el-form-item>

      <!-- 禁止查看完整信息 -->
      <el-form-item label="禁用状态-禁止查看">
        <vin-desensitize
          v-model="form.value2"
          s-type="phone"
          disabled
          :disabled-show="false"
        />
        <div class="demo-tip">
          禁止查看完整信息，不显示切换按钮
        </div>
      </el-form-item>

      <!-- 不同类型的禁用展示 -->
      <el-form-item label="银行卡号">
        <vin-desensitize
          v-model="form.value3"
          s-type="bank"
          disabled
        />
      </el-form-item>

      <el-form-item label="身份证号">
        <vin-desensitize
          v-model="form.value4"
          s-type="idCard"
          disabled
        />
      </el-form-item>

      <el-form-item label="邮箱地址">
        <vin-desensitize
          v-model="form.value5"
          s-type="email"
          disabled
        />
      </el-form-item>
    </el-form>
  </div>
</template>

<script>
export default {
  name: 'DesensitizeDisabled',
  data() {
    return {
      form: {
        value1: '13812345678',
        value2: '13812345678',
        value3: '6222021234567890123',
        value4: '310101199001011234',
        value5: 'example@domain.com'
      }
    }
  }
}
</script>

# 自定义脱敏规则

通过 desensitize 属性可以自定义脱敏处理规则。

保留首尾字符，中间用*替换
隔位显示*号，形成间隔效果
使用自定义字符#替���敏感信息

传入自定义的脱敏处理函数来实现特殊的脱敏需求。

<template>
  <div class="desensitize-demo">
    <el-form label-width="120px">
      <!-- 自定义脱敏规则示例1 -->
      <el-form-item label="保留首尾">
        <vin-desensitize
          v-model="form.value1"
          :desensitize="keepFirstAndLast"
          placeholder="保留首尾字符"
        />
        <div class="demo-tip">
          保留首尾字符，中间用*替换
        </div>
      </el-form-item>

      <!-- 自定义脱敏规则示例2 -->
      <el-form-item label="隔位脱敏">
        <vin-desensitize
          v-model="form.value2"
          :desensitize="alternateDesensitize"
          placeholder="隔位显示*号"
        />
        <div class="demo-tip">
          隔位显示*号，形成间隔效果
        </div>
      </el-form-item>

      <!-- 自定义脱敏规则示例3 -->
      <el-form-item label="自定义替换">
        <vin-desensitize
          v-model="form.value3"
          :desensitize="customReplace"
          placeholder="使用自定义字符替换"
        />
        <div class="demo-tip">
          使用自定义字符#替���敏感信息
        </div>
      </el-form-item>
    </el-form>
  </div>
</template>

<script>
export default {
  name: 'DesensitizeCustom',
  data() {
    return {
      form: {
        value1: '这是一段测试文本',
        value2: '13812345678',
        value3: '敏感信息示例'
      }
    }
  },
  methods: {
    // 保留首尾字符
    keepFirstAndLast(value) {
      if (!value) return '';
      const len = value.length;
      if (len <= 2) return value;
      return value.charAt(0) + '*'.repeat(len - 2) + value.charAt(len - 1);
    },
    // 隔位脱敏
    alternateDesensitize(value) {
      if (!value) return '';
      return value.split('').map((char, index) => 
        index % 2 === 0 ? char : '*'
      ).join('');
    },
    // 自定义替换字符
    customReplace(value) {
      if (!value) return '';
      return value.replace(/./g, '#');
    }
  }
}
</script>

# API

# 属性

参数 说明 类型 可选值 默认值
v-model 绑定值 string / number — —
s-type 脱敏类型 string phone / bank / idCard / email / all —
disabled 是否禁用 boolean — false
disabled-show 禁用状态下是否可以查看完整信息 boolean — true
desensitize 自定义脱敏处理函数 Function — —
placeholder 输入框占位文本 string — —
max-length 最大输入长度 number — —

# 事件

支持 Element UI Input 组件的所有事件

事件名 说明 回调参数
change 值改变时触发 (value: string)
input 输入时触发 (value: string)
blur 失去焦点时触发 (event: FocusEvent)
focus 获得焦点时触发 (event: FocusEvent)

# 插槽

支持 Element UI Input 组件的所有插槽

插槽名 说明
prefix 输入框头部内容
suffix 输入框尾部内容
prepend 输入框前置内容
append 输入框后置内容

# 脱敏规则说明

  • 手机号：保留前 3 位和后 4 位，中间用 * 代替
  • 银行卡号：保留前 4 位和后 4 位，中间用 * 代替
  • 身份证号：保留前 4 位和后 4 位，中间用 * 代替
  • 邮箱：保留 @ 前的第一个字符和最后一个字符，以及 @ 后的所有内容
  • 全部：除首尾字符外，其他字符全部用 * 代替
